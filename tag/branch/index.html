    Branch 分支

    分支让你从主线（如 main / master）分叉出去独立开发，完成后再合并回来。Git 的分支创建、切换、合并都非常轻量。

    分支查看

    git branch              # 本地分支，当前分支前有 *
git branch -v           # 附带最后一次提交信息
git branch -a           # 本地 + 远程分支
git branch -r           # 仅远程分支
git log --oneline --decorate --graph --all  # 图形化查看分叉历史
